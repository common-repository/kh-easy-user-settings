== Description ==

KH Easy User Settings is the perfect WordPress plugin for setting up user profiles. With this plugin, users can easily update their own details and you can choose to let them update a specific user meta data as well. Download this helpful plugin to make your user experience more efficient and enjoyable. KH Easy User Settings is the perfect solution for any website or blog that needs to manage user settings and profiles.

Here is a summary of what you can do:

*   A simple frontend form to allow user to update their details
*   Allows user to even update any custom user metadata from frontend
*   Avoid complicated or paid plugins that does the same thing.
*   And finally.. it's free!

If you need any assistance, feel free to contact me at [KnowHalim](https://knowhalim.com/contact-me "Contact Halim")

== Installation ==

To install, you just need to follow these steps

1. Upload `kh-easy-user-settings` zip file to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Navigate to Easy Frontend User Settings and configure.
